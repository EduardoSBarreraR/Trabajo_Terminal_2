\chapter{Objetivos}

\section*{Objetivo general}
Diseñar y construir un sistema robótico de recolección de muestras geológicas para un rover
de exploración, que sea capaz de obtener piedras del suelo para su posterior análisis y envío
de datos a una estación.


\section*{Objetivos Particulares.}


\begin{enumerate}
\item Trabajo Terminal I
	\begin{itemize}[noitemsep,nolistsep]
	
	\item Diseñar un módulo a manera de efector final 		que sea capaz de recoger muestras del
	suelo para su manipulación.
	
	\item Diseñar un sistema robótico de 6 grados de 		libertad que posicione y oriente el efector
	para transportar la muestra.

	\item Diseñar un módulo de percepción que le 			permita al sistema reconocer el ambiente
	para crear un marco de referencia en el espacio.

	\item Diseñar un sistema de procesamiento y 			comunicación que permita el análisis de las
	muestras para determinar las características del 		entorno, además de permitir la
	transmisión y recepción de datos. 

	\item Integrar los módulos de manera computacional 	para que trabajen en conjunto.

	\item Validar el sistema mecatrónico mediante 			simulaciones con el objetivo de comprobar
	que los parámetros obtenidos cumplen su función.
	
	\end{itemize}

\item Trabajo Terminal II
	\begin{itemize}[noitemsep,nolistsep]
	
	\item Implementar el módulo de efector final que 		sea capaz de recoger muestras del suelo
	para su manipulación.
	
	\item Implementar un sistema robótico de 6 grados 		de libertad que posicione y oriente el
	efector para transportar la muestra.

	\item Implementar un módulo de percepción que le 		permita al sistema reconocer el
	ambiente para crear un marco de referencia en el 		espacio, así como identificar la
	geometría de la muestra.

	\item Implementar un sistema de procesamiento y 		comunicación que permita el análisis de
	las muestras para determinar las características 		del entorno, además de permitir la
	transmisión y recepción de datos.

	\item Realizar pruebas y ajustes necesarios para 		garantizar que cada módulo trabaja
	correctamente de manera independiente.

	\item Integrar secuencialmente los módulos para 		consolidar el sistema mecatrónico.

	\item Verificar que todos los módulos del sistema 		mecatrónico trabajen armónicamente para
	posteriormente realizar pruebas de recolección de 		muestras en un ambiente análogo
	que nos permitan comprobar que el sistema cumple 		con la función principal.
	
	\end{itemize}
\end{enumerate}
